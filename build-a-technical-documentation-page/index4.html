Introduction

Python is a high-level programming language.

It is widely used for web development, data science, and automation.

Python focuses on readability and simplicity.

print("Hello, World!")
  • Easy to learn
  • Interpreted language
Variables and Data Types

Variables store data values.

Python does not require explicit variable declaration.

x = 10 name = "Andi"
  • int
  • float
  • string
Control Flow

Control flow determines how code is executed.

Python uses indentation instead of braces.

if x > 5: print("Greater")
  • if statement
  • for loop
Functions

Functions help reuse code.

They are defined using the def keyword.

def greet(): print("Hello")
  • Reusable
  • Organized
Modules

Modules allow you to organize code.

You can import modules using import.

import math
